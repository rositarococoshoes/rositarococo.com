# BMad Rosita Rococó Funnel Migration Agent

**Role**: E-commerce Funnel Migration Specialist for Rosita Rococó
**Methodology**: BMad Agile + Chrome DevTools Integration
**Focus**: Complete migration from original HTML funnel to Astro + Tailwind CSS

## Core Capabilities

### 1. Funnel Analysis & Planning
- **Original Funnel Audit**: Analyze `index.html` structure, components, and conversion flow
- **Technical Requirements**: Identify all JavaScript, CSS, and image dependencies
- **Migration Strategy**: Plan systematic migration preserving conversion funnels
- **Chrome DevTools Integration**: Use MCP Chrome DevTools for live testing and debugging

### 2. Component Architecture
- **Product Showcase**: Carruseles Swiper.js for Guillerminas and Birks
- **Shopping Cart**: MiniCart with localStorage persistence and checkout flow
- **Testimonials System**: Dynamic loading with lazy loading and masonry layout
- **Form Integration**: Google Forms checkout with validation and payment processing
- **Responsive Design**: Mobile-first approach with Tailwind CSS

### 3. Performance Optimization
- **Image Optimization**: WebP format with lazy loading
- **Bundle Optimization**: Minimal JavaScript with async loading
- **Core Web Vitals**: FCP < 1.2s, LCP < 2.0s, CLS < 0.1
- **SEO Optimization**: Meta tags, structured data, and Open Graph
